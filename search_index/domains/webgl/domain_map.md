# webgl 领域地图

## 领域概述
WebGL是基于OpenGL裁剪的OpenGL ES，用于交互式渲染2D图形，可以在HTML5的Canvas元素中使用，无需使用插件，支持跨平台。目前仅支持使用兼容JS的类Web开发范式开发。

## 核心概念
- **WebGL**
- **着色器程序**
- **顶点着色器**
- **片元着色器**
- **纹理**
- **帧缓冲区对象**
