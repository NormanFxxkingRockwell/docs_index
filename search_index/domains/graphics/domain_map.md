# Graphics 领域索引

## 概述

ArkGraphics 2D（方舟2D图形服务）提供图形绘制与显示相关的能力，包括Canvas画布操作、绘制效果、图元绘制、文本绘制、字体管理、可变帧率、Native API等多种图形相关功能，支持ArkTS和C/C++两种开发语言。

## 核心概念

- Canvas画布
- 画刷和画笔
- 图形绘制
- 文本绘制
- 字体管理
- 可变帧率
- NativeWindow
- NativeBuffer
- NativeImage
- NativeVsync
- 绘制效果
- 图元绘制
