# mechanicManager 领域地图

## 领域概述
Mechanic Kit是机械体设备控制器提供的API集合，从API version 20起，应用可使用Mechanic Kit控制机械体设备，提供设备连接管理、目标跟踪和状态监控等功能。Mechanic Kit通过mechanicManager模块，实现设备连接管理、目标跟踪和状态监控，支持精准控制，帮助开发者构建高质量的设备交互应用。

## 核心概念
- **机械体设备控制**
- **目标跟踪**
- **设备转动控制**
- **设备连接管理**
- **状态监控**
