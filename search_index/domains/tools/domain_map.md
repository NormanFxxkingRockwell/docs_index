# Tools 领域地图

## 领域概述

Tools领域涵盖了OpenHarmony应用开发过程中的各种命令行工具和调试工具，包括应用打包拆包、安装管理、调试测试、权限管理、账号管理、通知管理、资源编译、媒体库操作、网络配置、电源管理、系统参数管理、企业设备管理、二进制签名、SDK升级助手等23个工具，为开发者提供从应用构建到调试测试的全流程支持。

## 核心概念

- 应用打包拆包
- 应用安装管理
- 应用调试
- 权限管理
- 账号管理
- 通知管理
- 资源编译
- 媒体库操作
- 网络配置
- 电源管理
