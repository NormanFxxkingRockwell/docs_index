# Tools 领域

## 领域概述

Tools 领域包含 OpenHarmony 开发过程中使用的各种命令行工具和调试工具，涵盖应用调试、包管理、资源编译、网络配置、系统参数操作、电源管理、媒体资源访问、通知管理、账号管理、访问控制管理、企业设备管理、SDK升级辅助、二进制签名、堆快照分析等多种开发辅助工具。

## 核心概念

- 应用调试工具
- 包管理工具
- 资源编译工具
- 命令行工具集合
- 系统参数操作
- 电源状态管理
- 媒体资源访问
- 通知管理工具
- 账号管理工具
- 访问控制管理
- 企业设备管理
- SDK升级辅助
