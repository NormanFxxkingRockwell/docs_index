# ArkTS 领域地图

## 概述

ArkTS是OpenHarmony应用开发的官方高级语言，在TypeScript生态基础上做了进一步扩展。ArkTS基础类库提供XML生成解析转换、Buffer与FastBuffer、容器类库、JSON扩展等能力。ArkTS并发编程提供TaskPool和Worker两种多线程并发API，以及Sendable对象支持跨线程引用传递。ArkTS运行时支持解释了器、AOT和JIT高效执行方式，提供GC垃圾回收、模块化运行、动态加载等能力。ArkTS编译工具链支持语法检查、UI转换、源码混淆、字节码编译、反汇编等功能。

## 核心概念

- ArkTS编程语言
- TaskPool多线程
- Worker线程
- Sendable对象
- 并发编程
- 容器类库
- XML处理
- Buffer二进制
- GC垃圾回收
- 模块化运行
