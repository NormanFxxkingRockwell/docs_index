# Graphics3D 领域地图

## 概述

ArkGraphics 3D（方舟3D图形）是HarmonyOS 3D图形渲染的核心模块，基于轻量级的3D引擎以及渲染管线为开发者提供基础3D场景绘制能力。支持场景搭建、模型加载、相机管理、光源配置、材质系统、着色器、图片资源（PNG/JPG/KTX）、环境资源和动画控制等功能，采用基于物理的渲染（PBR）模型，提供便捷、高效的3D场景构建和渲染能力。

## 核心概念

- **3D场景**：由光源、相机、模型三个关键部分组成的渲染环境
- **模型加载**：支持glTF格式的3D模型加载和呈现
- **相机管理**：提供观察视角的创建、配置和交互控制
- **光源配置**：提供光照效果的创建、配置和属性调整
- **材质系统**：基于物理的渲染（PBR）模型，定义物体表面视觉效果
- **着色器**：支持自定义Shader和PBR材质两种类型
- **图片资源**：支持PNG、JPG、KTX格式的纹理图片
- **环境资源**：支持立方体和球体贴图的环境背景
- **动画控制**：提供场景动画的播放和控制能力
- **PBR渲染**：基于物理的渲染模型，包含金属度、粗糙度等属性
