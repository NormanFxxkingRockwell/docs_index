# Security 领域地图

## 概述

Security（安全）是OpenHarmony提供的综合安全服务框架，涵盖访问控制、应用权限管理、用户认证、密钥管理、加解密算法、数据保护、证书管理、关键资产存储、安全控件和应用签名等功能。通过程序访问控制（AccessToken）、Asset Store Kit、Crypto Architecture Kit、Data Protection Kit、Device Certificate Kit、Universal Keystore Kit、User Authentication Kit等子模块，为应用提供完整的安全防护能力。

## 核心概念

- 访问控制
- 应用权限
- 用户认证
- 密钥管理
- 加解密
- 数据保护
- 证书管理
- 关键资产存储
- 安全控件
- 应用签名
