# test 领域地图

## 领域概述
Test Kit 提供应用测试相关功能，包括自动化测试框架（单元测试、UI测试、白盒性能测试）和命令行工具（SmartPerf性能工具、wukong稳定性工具）。

- **UI测试**
- **白盒性能测试**
