# Media 领域地图

## 领域概述
Media Kit 提供音视频媒体处理能力，包括音频播放和录制、视频编解码、相机开发、图像处理、DRM数字版权保护、媒体会话管理、屏幕录制等功能，支持多种媒体格式和场景化应用。

## 核心概念
- **图像处理**
- **媒体会话管理**
- **相机开发**
- **音频处理**
- **音视频编解码**
- **视频处理**
- **媒体录制**
- **DRM数字版权**
- **屏幕录制**
- **媒体播放**
