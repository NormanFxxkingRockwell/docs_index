# Reference 领域地图

## 概述
API 参考领域包含 3295 个文档，涵盖 OpenHarmony 系统的各类 API 接口文档。 主要包括 Ability Kit, Basic Services Kit, Image Kit, Media Kit, Camera Kit 等模块的 API 参考。 提供 ArkTS 和 C/C++ 两种开发语言的接口说明，包括类、接口、枚举、错误码等详细文档。

## 核心概念
- Native
- API参考
- 接口
- 类
- API
- 模块
- 错误码
- ArkTS
- 枚举
